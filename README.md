# Car-Rental-System - OOPS
# 🚗 Car Rental System (Java Console Application)

A simple **Object-Oriented Java console application** that simulates a **Car Rental System**.  
Users can rent available cars, return rented cars, and view rental details through a menu-driven interface.

This project demonstrates **core Java concepts such as OOP, encapsulation, collections, and basic system design**.

---

## 📌 Features

- View available cars
- Rent a car
- Return a rented car
- Customer management
- Rental record tracking
- Menu-driven console interface

---

## 🛠 Technologies Used

- **Java**
- **Object-Oriented Programming (OOP)**
- **Java Collections (ArrayList)**
- **Scanner (User Input)**

---

## 📂 Project Structure

### Class Description

| Class | Description |
|------|-------------|
| `Car` | Stores car details and availability |
| `Customer` | Stores customer information |
| `Rental` | Links a customer with a rented car |
| `CarRentalSystem` | Manages cars, customers, rentals, and menu |
| `Main` | Entry point of the program |

---

## 🚘 System Workflow

1. User selects **Rent a Car**
2. System displays **available cars**
3. User enters:
   - Name
   - Car ID
   - Rental days
4. System calculates **total price**
5. User confirms rental
6. Car becomes **unavailable**
7. User can later **return the car**

---

## 💰 Price Calculation

Each car has its own **base rental price per day**.
